# 🧪 Machine Learning Experiments – IML Lab

## 🔎 Introduction
This repository showcases practical experiments from the **Introduction to Machine Learning Lab**.  
It highlights how fundamental ML concepts can be applied using Python, with a focus on hands-on learning and clear explanations.

---

## 🎯 Goals
- Explore essential machine learning workflows  
- Apply preprocessing techniques to real datasets  
- Implement and compare ML algorithms  
- Measure accuracy and performance of models  

---

## ⚙️ Tools & Libraries
- **Core Language:** Python  
- **Environment:** Jupyter Notebook  
- **Libraries:** NumPy, Pandas, Matplotlib, Scikit-learn  

---

## 📂 Repository Contents
- Jupyter notebooks with step-by-step implementations  
- Sample datasets for training and evaluation  
- Documentation and notes for each experiment  

---

## ▶️ How to Use
1. Clone this repository to your local machine  
2. Launch Jupyter Notebook in the project directory  
3. Open any experiment notebook and run cells sequentially
